# LaudBot — Brand

## Palette

| Role | Name | Hex |
|---|---|---|
| Primary | Navy | `#1e3a8a` |
| Primary dark (hover) | Navy dark | `#1e3366` |
| Background | White | `#ffffff` |
| Surface | Slate 50 | `#f8fafc` |
| Text primary | Slate 900 | `#0f172a` |
| Text secondary | Slate 600 | `#475569` |
| Text muted | Slate 400 | `#94a3b8` |
| Border | Slate 200 | `#e2e8f0` |
| Dark background | Zinc 950 | `#09090b` |
| Dark surface | Zinc 900 | `#18181b` |

Three-colour brand: **black · white · navy.**

## Typography

- Font stack: `-apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif`
- No custom font loaded — system fonts keep the UI fast and native-feeling

## Voice

- Terse. No filler. No hype.
- Confident, not salesy.
- Mode-dependent: professional → direct and grounded; peer → candid and technical; buddy → dry and self-aware.

## Logo

- Wordmark only: **LaudBot**
- No icon/logomark yet
- Casing: title case — LaudBot, not laudbot or LAUDBOT

## Email

- Light layout — white background, navy CTA
- Never dark-mode email — too many client rendering issues
- See `backend/app/services/email.py` for the HTML template
